# Xplorer


![img](img/explorer.png)


**1. INstall virtual env**
```
pip install virtualenv
```

**2. GO in to the flask repo and create virtual env**

```
cd flaskapp
virtualenv venv
```

**3. Activate the virtual environment**
```
venv/bin/activate
```

**4. Install Flask**
```
pip install Flask
```

**5. Run the application**
```
python icd.py
```
